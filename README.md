# Clinic Management System Database

A database project designed for managing clinic operations including patients, doctors, consultations, hospitalisation, prescriptions, billing, and medical records.

## Project Components
- ER Diagram (Draw.io)
- Relational Model
- SQL Table Creation Script

## Technologies Used
- SQL
- Draw.io
- Database Design Concepts

## Core Entities
- Patient
- Doctor
- Nurse
- Consultation
- Prescription
- Invoice
- Insurance

- Primary and foreign keys
- SQL table creation
